# 论文AI利器

论文写作辅助平台，一站式搞定 DOCX 解析、多源文献检索、参考文献管理、知识图谱、论文分析。

## 功能模块

| 模块 | 说明 |
|------|------|
| ✅ 格式检查 | 标题层级、图表编号连续性、参考文献格式完整性 |
| 🔤 术语分析 | 高频术语提取、同义表述检测、中英混用、缩写检查 |
| 📝 段落分析 | 段落长度分布、过渡词、可读性评分 |
| 💡 优化建议 | 结构诊断、文献密度、摘要质量评估、句式检查 |
| 🕸️ 知识图谱 | 力导向网络图 + 交互式词云 + 文献时间线 |
| 📋 参考文献 | 多源检索、自动注入、GB/T 7714/BibTeX 导出 |

## 快速开始

### 环境要求
- Python 3.8+
- 依赖: `pip install flask requests`

### 启动
```bash
# Windows: 双击 启动.bat
# macOS/Linux:
python start.py
```
浏览器自动打开 `http://localhost:5000`

### 使用流程
1. 上传 .docx 论文
2. 切换模块标签进行各项分析
3. 参考文献模块中点击"检索文献"自动搜索并注入
4. 知识图谱查看词云/网络图/时间线

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| Ctrl+1~6 | 切换模块 |
| Ctrl+Enter | 检索文献 |
| Ctrl+B | 导出参考文献 |
| Ctrl+O | 换论文 |
| Esc | 关闭弹窗 |
